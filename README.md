# NEWSPULSE_PRO
An AI-powered news analytics dashboard with sentiment analysis,TF-IDF, LDA, NER, stock correlation and role-based login. Built with MediaStack API and Streamlit.

🌐 NewsPulse Pro — AI News Analytics Dashboard

By Mithila Dhar

📰 Overview
NewsPulse Pro is an AI-powered news analytics dashboard that collects real-time global news using the MediaStack API and analyses trending topics and sentiment using NLP techniques including TF-IDF, LDA, TextBlob and spaCy.

✨ Features

🔐 Role-based login — Admin & User with manual registration (no hardcoded credentials)

📡 Real-time news from MediaStack API (50+ countries, 7 categories)

😊 Sentiment Analysis using TextBlob (Positive / Negative / Neutral)

☁ Word Cloud coloured by TextBlob polarity

🔤 TF-IDF keyword extraction with 4 view modes

🧠 LDA Topic Modelling (5 topics)

🗺 Named Entity Recognition using spaCy

📈 Stock market × sentiment correlation (Yahoo Finance)

🤖 AI article summariser (TextRank — no API needed)

📊 ML models: Logistic Regression, SVM, Naive Bayes

👑 Admin: user management, password reset, raw data, CSV export

👤 User: news dashboard, sentiment, keywords, NER, stock
